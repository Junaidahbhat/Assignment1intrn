\textbf{Assume X,Y,Z,W and P are matrices of orders
2×n,3×k,\\
2×p,n×3 and p×k,respectively.\\
The restriction on n,k and p so that PY+WY will be defined are:\\
(A)k=3,p=n\\
(B)k is arbitrary,p=2\\
(C)p is arbitrary,k=3\\
(D)k=2,p=3\\
}
\\
\begin{large}
Solution:\\
Concept 1 :The product of two matrices is defined only if the  number of columns in the first matrix is equal to the number of rows in the second matrix and the resulting matrix will have the same number of rows as the first matrix and the same number of columns as the second matrix.\\ \\
Concept 2 :To add two matrices,the dimensions of both must be same,that is, they must have same number of rows and columns.\\  \\
Now getting back to question,\\

\pagebreak

Matrices P and Y are of the order \textit{p$\times$k} and \textit{3$\times$k} respectively.\\

\textit{PY=}[P]\begin{scriptsize}
\textit{p$\times$k}
\end{scriptsize}
[Y]\begin{scriptsize}
\textit{3$\times$k}
\end{scriptsize}
.\\

By concept 1,it is possible only if\\

\textit{ k=3}.\hspace{6cm}               (1)\\

and its order will be \textit{p$\times$k},So 
\textit{PY}\textit{\begin{scriptsize}
p$\times$k
\end{scriptsize}}\\       

Simlarly,\\
Matrices W and Y are of the order n$\times$3 and 3$\times$k respectively.\\

\textit{WY=}[W]\begin{scriptsize}
\textit{n$\times$3}
\end{scriptsize}
[Y]\begin{scriptsize}
\textit{3$\times$k}
\end{scriptsize}
.\\

By concept 1,it is possible beacause no. of colums of first matrix equals to number of rows of second matrix.\\

and its order will be \textit{n$\times$k},So
\textit{WY}\textit{\begin{scriptsize}
n$\times$k
\end{scriptsize}}\\

\textbf{Now adding},\\

\textit{PY}\begin{scriptsize}
\textit{p$\times$k}
\end{scriptsize}
\textit{+WY}\begin{scriptsize}
\textit{n$\times$k}
\end{scriptsize}\\ 

By concept 2,it is possible only if
\\

\textit{p$\times$k = n$\times$k}\\

i,e \textit{p=n} \hspace{6cm}    (2)\\

Therefore from eqn(1) and (2) \textbf{Option A is correct}
